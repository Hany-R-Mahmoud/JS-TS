# Project Title

        Memory Block Game

## Description

A Memory Game, following tutorials from "Elzero Web School" on YT by Osama Elzero

## Features

- interactive memory game with multiple blocks.
- counting trial times to finish all.
- random card poisitions with each refresh.

## Technologies Used

- HTML
- CSS / SASS
- JavaScript
