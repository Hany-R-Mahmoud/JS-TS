# Project Title

        Hangman game

## Description

A game for guessing words, following tutorials from "Elzero Web School" on YT by Osama Elzero

## Features

- interactive game of guessing words.
- multiple categories, with different words in each category.

## Technologies Used

- HTML
- CSS / SASS
- JavaScript
